## PDF Export

To export any guide as PDF:

1. Open the guide in your browser (or GitHub)
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
3. Select "Save as PDF" as printer
4. The Trading Department branding will be included

Alternatively, open `generate-pdf.html` in a browser for a branded index page.
